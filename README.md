## Commands 
`--setvolume` or `-sv`
- Eg: `lgtv --setvolume 35` 

`--turnoff` or `-to`
- Eg: `lgtv --turnoff`

`--message` or `-m`
- Eg: `lgtv --message "Hello World"`


## Development 
- Use `npm link` such that you can use `lgtv` instead of `./app.js`
